Engine:  https://github.com/3b1b/manim	"3b1b/manim"

Video Source File: aptch.py

---

## Build

```bash
manim aptch.py -a
```

Flags:

`-a`: build all videos

`-p`: preview

`-l`: low quality (15 frames/sec)

`-m`: medium quality (30 frames/sec)

`-s`: build only the last image
